## Getting Started With Project Set-up

### Created Using
- Next JS
- Tailwind CSS
- Lucide Icons
- AOS
- Marquee

### Installation & Setup

1. Clone the repository:
   
   ```bash
   git clone https://github.com/harshaltupe12/mugs.git
   ```

2. Navigate to the project directory
   ```bash
   cd your-repo-name
   ```

3. Install dependencies 
   ```bash
   npm install
   ```

4. Run the project locally 
   ```bash
   npm run dev
   ```
